About this Algorithm

Bubble Sort

Bubble Sort compares adjacent elements and swaps them if they're in the wrong order. The largest unsorted element "bubbles up" to its correct position on each pass.

Time Complexity
Best O(n)
Average O(n²)
Worst O(n²)
Space O(1)
Properties
Stable
In-Place
Not Adaptive
